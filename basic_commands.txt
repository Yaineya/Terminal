Ls  - просмотр файлов папки по алфавиту без данных о файлах

Ls -la - просмотр файлов папки по алфавиту с данных о файлах, с ролями, размером, датой создания

Cd - сменить директорию
Cd ../ - выход на директорию выше
Cd ../../ - выход на 2 директории выше
. - указатель 
.. - указатель на папку в директории выше

Mkdir [название папки без скобок] - создание папки в директории
Cd [название папки без скобок]/ - войти в указанную папку/файл

Touch [название файла].формат файла - создание файла
Cat [название файла] - показать содержимое файла

CTRL + C  - закрыть редактирование в терминале

Vim [название файла] - зайти в файл для редактирования
Жмем I (insert) для начала редактирования
Eng раскладка! Жмем Esc, далее комбинацию ниже
:q - выйти из vim если не вносил изменения
:wq - выйти из vim и сохранить изменения
:q! - выйти из vim и не сохранять изменения

Rm test.txt - удаляет файл
Rm -I test.txt - удаляет файл, но нужно подтвердить
Rm -r  -удаляет папку и все вложенные папки (рекурсивно) 
Rmdir [название папки] - удалит папку, если она пустая

Grep
Команда grep служит для поиска строк, содержащих заданный пользователем образец
Заключаем в кавычки, можно слово, можно часть.
Шаблон:
Grep ‘образец’  [в каком файле/файлах через пробел]
Флаги: -r recursive рекурсивно обследовать все дерево указанной директории
-i ignore-case игнорировать регистр
-c count подсчитывает количество строк, в которых обнаружен образец
-n line-number номера строк
-v invert-match выводит строки, в которых образец не встречается
-w word-regexp искать только строки, содержащие все слово или фразу, составляющую образец
-x отберет только те строки исследуемого файла или файлов, которые полностью совпадают с образцом
-l сообщает лишь имена файлов, в которых данный образец найден
-L сообщает имена тех файлов, где не встретился образец

grep 'ar' 22.txt /dev/null: - выведет файл:совпадение
